# Руководство пользователя (RU)

## 1. Введение
Это руководство объясняет, как использовать веб-приложение «Escoles Publiques».

Целевая аудитория:
- администраторы (`ADM`)
- конечные пользователи (`USER`) (ограниченный доступ)

Требования:
- современный браузер
- действующие учетные данные

## 1.1 Защита данных (важно)
- Не включайте реальные персональные данные (имена/e-mail/ID) в документацию.
- Для скриншотов используйте тестовые данные и скрывайте/размывайте чувствительную информацию перед публикацией.
- В репозитории не должно быть скриншотов с реальными данными.

## 2. Вход
1. Откройте страницу входа.
2. Введите e-mail и пароль.
3. Нажмите «Войти».

Скриншоты:
![Вход (RU)](../codex_images_real/en/login-admin.png)

## 3. Навигация
Главное меню администратора:
- Главная, Школы, Ученики, Зачисления, Годовые взносы

Выбор языка:
- выберите язык в верхней панели (сохраняется в cookie)

## 4. Школы (ADM)
- список/поиск/сортировка
- создание/редактирование/удаление
- избранное и назначение области (scope)

## 5. Ученики (ADM)
- создание/редактирование/удаление
- повторное использование пользователя по e-mail

## 6. Зачисления (ADM)
- создание/редактирование/удаление зачисления (учебный год, курс, статус)

## 7. Годовые взносы (ADM)
Создание:
1. Перейдите в раздел «Годовые взносы».
2. Нажмите «Новый» (откроется модальное окно).
3. Выберите зачисление, введите сумму и дату платежа.
4. Подтвердите чекбокс конфиденциальности и сохраните.

Десятичные значения:
- сумма поддерживает `1000,25` и `1000.25`
