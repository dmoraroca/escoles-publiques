# Manuel utilisateur (FR)

## 1. Introduction
Ce manuel décrit l'utilisation de **Escoles Publiques** étape par étape, écran par écran.

Profils :
- `ADM` : administration complète
- `USER` : accès limité au tableau de bord

## 2. Connexion
Étapes :
1. Ouvrez l'écran de connexion.
2. Saisissez l'e-mail et le mot de passe.
3. Cliquez sur **Connexion**.

![Connexion (FR)](/ajuda/codex_images_real/en/login-admin.png)

## 3. Écran d'accueil
### 3.1 Partie supérieure
Contenu :
- en-tête avec menu et sélecteur de langue
- barre de recherche principale

![Accueil haut (FR)](/ajuda/codex_images_real/en/inici-top.png)

### 3.2 Partie inférieure
Contenu :
- cartes de périmètre
- pied de page et liens utiles

![Accueil bas (FR)](/ajuda/codex_images_real/en/inici-down.png)

## 4. Écoles
### 4.1 Liste
Étapes :
1. Ouvrez **Écoles**.
2. Filtrez via la recherche.
3. Vérifiez les résultats et les actions disponibles.

![Écoles liste (FR)](/ajuda/codex_images_real/en/escoles.png)

### 4.2 Créer une école
Étapes :
1. Cliquez sur **Nouvelle école**.
2. Renseignez les champs obligatoires.
3. Enregistrez.

![Écoles créer (FR)](/ajuda/codex_images_real/en/escoles-crear.png)

### 4.3 Détail d'une école
Étapes :
1. Cliquez sur **Détails**.
2. Contrôlez les informations complètes.

![Écoles détail (FR)](/ajuda/codex_images_real/en/escoles-detall.png)

### 4.4 Modifier une école
Étapes :
1. Cliquez sur **Modifier**.
2. Ajustez les champs nécessaires.
3. Enregistrez.

![Écoles modifier (FR)](/ajuda/codex_images_real/en/escoles-edit.png)

### 4.5 Supprimer une école
Étapes :
1. Cliquez sur **Supprimer**.
2. Confirmez l'action.

![Écoles supprimer (FR)](/ajuda/codex_images_real/en/escoles-esborrar.png)
