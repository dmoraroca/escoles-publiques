# Manual d'usuari (CA)

## 1. Introduccio
Aquest manual explica l'ús de la web **Escoles Publiques** pas a pas, pantalla a pantalla.

Perfils:
- `ADM`: administracio completa
- `USER`: accés limitat al tauler

## 2. Inici de sessio
Què fer:
1. Obre la pantalla de login.
2. Introdueix correu i contrasenya.
3. Prem **Entrar**.

![Login (CA)](/ajuda/codex_images_real/ca/login-admin.png)

## 3. Pantalla d'inici
### 3.1 Part superior
Què hi ha:
- capçalera amb menu i canvi d'idioma
- cercador principal

![Inici top (CA)](/ajuda/codex_images_real/ca/inici-top.png)

### 3.2 Part inferior
Què hi ha:
- targetes d'ambit
- zona inferior amb peu i accessos

![Inici down (CA)](/ajuda/codex_images_real/ca/inici-down.png)

## 4. Escoles
### 4.1 Llistat
Què fer:
1. Ves a **Escoles**.
2. Filtra amb el cercador.
3. Revisa resultats i accions.

![Escoles llistat (CA)](/ajuda/codex_images_real/ca/escoles.png)

### 4.2 Crear escola
Què fer:
1. Prem **Nova escola**.
2. Omple camps obligatoris.
3. Desa.

![Escoles crear (CA)](/ajuda/codex_images_real/ca/escoles-crear.png)

### 4.3 Detall d'escola
Què fer:
1. Prem **Detalls** a una fila.
2. Revisa informació completa.

![Escoles detall (CA)](/ajuda/codex_images_real/ca/escoles-detall.png)

### 4.4 Editar escola
Què fer:
1. Entra a **Editar**.
2. Modifica camps.
3. Desa canvis.

![Escoles editar (CA)](/ajuda/codex_images_real/ca/escoles-edit.png)

### 4.5 Eliminar escola
Què fer:
1. Prem **Esborrar**.
2. Confirma l'acció.

![Escoles esborrar (CA)](/ajuda/codex_images_real/ca/escoles-esborrar.png)
