# Руководство пользователя (RU)

## 1. Введение
Это руководство описывает работу с **Escoles Publiques** пошагово, по каждому экрану.

Профили:
- `ADM`: полный административный доступ
- `USER`: ограниченный доступ к панели

## 2. Вход в систему
Шаги:
1. Откройте экран входа.
2. Введите e-mail и пароль.
3. Нажмите **Войти**.

![Вход (RU)](/ajuda/codex_images_real/en/login-admin.png)

## 3. Главная страница
### 3.1 Верхняя часть
Содержимое:
- шапка с меню и переключателем языка
- основная строка поиска

![Главная верх (RU)](/ajuda/codex_images_real/en/inici-top.png)

### 3.2 Нижняя часть
Содержимое:

- нижний колонтитул и ссылки

![Главная низ (RU)](/ajuda/codex_images_real/en/inici-down.png)
